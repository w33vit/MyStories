### Feature
- Note message.
- You can use voice to typing message. (Support Thai & English only)

### 3rd party lib
- Android Design Support Library
- UltimateRecyclerView

- Parceler

### Project requires
- API min 15 and target 22
- If you use voice typing, you must be internet connection.
